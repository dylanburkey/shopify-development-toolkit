# Project Overview

## Purpose
This is a **Shopify Theme Builder** application that allows users to create, customize, and export Shopify themes. It provides:
- A visual interface for building Shopify themes
- A library of pre-built sections, blocks, and components
- Custom section variants and presets (color/typography)
- Theme export functionality (ZIP files)
- Real-time Liquid template preview

## Target Platform
Deployed to **Cloudflare Workers** as a serverless application with edge computing capabilities.

## Key Features
- Browse and select from a library of theme sections and blocks
- Create and manage theme projects
- Apply color and typography presets
- Generate complete Shopify theme files (Liquid, JSON, assets)
- Export themes as ZIP files ready for Shopify deployment
- Drag-and-drop interface for organizing sections and blocks
